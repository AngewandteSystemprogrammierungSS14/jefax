Detailed Description

This file contains the messageQueue buffer type and functions used for USART communication and the jefax shell.

It uses the dynamic memory management for buffer memory allocation. Heap utilization:

  • 6 Byte messageQueue overhead + 1 memory management
  • 8 Byte message overhead + 1 memory management

Example: 1 messageQueue with 2 messages (10 and 20 Byte): Queue (6 + [1]) + message1 (8 + 10 + [2]) + message2 (8 + 20 + [2]) = 57 Byte (Overhead from memory management is in squared brackets)
